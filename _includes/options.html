Edition Used for Search and Display

Original versions aim to be exact reproductions of the copytexts, except for the addition of some helpful metadata (e.g. paragraph ids and footnote numbers) and some insubstantial formatting changes. Edited versions include some additional changes, either sanctioned by Hume himself or felt necessary by us for one reason or another. See the Notes for more details.

Further Display Options

When displaying the edited text, you can optionally show changes; these will appear with the original text crossed out, and the inserted text underlined. Hovering over the change will then bring up a tooltip briefly explaining the intervention. For paragraphs that span more than one page, page breaks can optionally be displayed (they will show up as a pipe symbol “|”).
